# Go by Example – My Learning Journey 🚀

This repository contains my practice solutions for **Go by Example**.
I am using this repository to learn the **Go (Golang) programming language** step by step by writing code, experimenting, adding bugs, fixing them, and understanding how things work internally.

The goal of this repository is to build a **strong foundation in Go** by practicing real examples.

---

## 📚 Source

All examples are inspired by the official learning resource:

Go by Example
https://gobyexample.com/

---

## 🎯 Learning Approach

For each topic, I follow this learning process:

1. **Learn the Concept**
2. **Write the Example Code**
3. **Add Bugs Intentionally**
4. **Fix the Bugs**
5. **Explain the Code Flow**

This helps me understand the concept deeply instead of just copying the code.
